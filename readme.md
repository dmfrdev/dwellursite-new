# dwellur premium static landing page — larger image

This version keeps the same overall layout and image placement, but gives the hero photo more room so you can see more of it.

## Deploy
1. Unzip this folder.
2. Go to https://app.netlify.com/drop
3. Drag the entire `dwellur-premium-site-larger-image` folder onto the page.

## Customize
- Edit words in `index.html`
- Edit colors/spacing/layout in `styles.css`
- Replace `dwellur-hero.jpg` with a new image using the same filename
